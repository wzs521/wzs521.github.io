分治算法（Shellsort）

汉诺塔游戏的演示和思路分析：

1.如果是有一个盘，A->C

如果我们有n>=2情况，我们总是可以看作是两个盘，最下边的一个盘，上面的所有盘子

（1）先把最上面的盘A>-B

（2）把最下面的盘A>-C

（3）把B塔的所有盘从B->C

代码实现

public class adsa {

public static void main(String[] args) {
    // TODO Auto-generated method stub
    adsa(5,'A','B','C');

}
public static void adsa(int num,char a,char b,char c) {
    if(num==1) {
        System.out.println("第一个盘"+a+"->"+c);
    }else {
 //             如果我们有n>=2情况，我们总是可以看作是两个盘，最下边的一个盘，上面的所有盘子
//            （1）先把最上面的盘A>-B,移动过程中会使用到C
        adsa(num-1, a, c, b);
//            （2）把最下面的盘A>-C
        System.out.println("第"+num+"个盘从"+a+"->"+c);
//            （3）把B塔的所有盘从B->C
        adsa(num-1, b, a, c);
      }
  }

 }
