Git

一.基本信息设置

1.设置用户名

git config –global user.name ‘was521’

2.设置用户名邮箱

git config –global user.email ‘2267493405@qq.com’

3.初始化

创建一个git仓库，创建之后就会在当前目录生成一个.git的文件

git init

二.向仓库添加文件（三步骤）

1.创建文件

在所在文件夹创建文件
touch 名称.文件类型
查看状态
git status

2.把文件从工作区提交到暂存区

git add 文件名.文件类型
查看状态，是否成功
git status

3.把文件从暂存区提交到仓库

git commit -m ‘描述 名称.文件类型’
查看是否成功
git status
